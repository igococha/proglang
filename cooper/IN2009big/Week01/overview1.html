Module IN2006 - Functional Programming
Title Week 1 - Introduction to Functional Programming
Maintainer Christian Cooper (Original author: Ross Paterson)

Week Aims and Instructions

The aim of this session is to introduce functional programming, and to get you started with the Hugs interpreter for the Haskell language. On completion, you should be able to write your own simple functions in Haskell and execute them in Hugs.
